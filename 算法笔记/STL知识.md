## STL知识总结

#### String类相关

```c++
#include <string>
string s;
string a = "aaa", b="bbb";
s.clear();  // clear string
s.size();   // size of string.
s.length(); // equal to s.size().
s.empty();  // judge whether the string is null.
s.find(substring); // find index of the first occurrence of the substring.
a.swap(b); // swap string a and b.
```
